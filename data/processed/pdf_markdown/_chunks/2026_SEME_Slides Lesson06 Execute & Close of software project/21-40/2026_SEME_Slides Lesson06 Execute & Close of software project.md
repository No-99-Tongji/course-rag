## Schedule Performance Index (SPI)

A measure of how close the project is to performing work as it was actually scheduled.

$$
\mathsf { S P } | = \mathsf { E V } / \mathsf { P V }
$$

<!-- image-->

<!-- image-->

## Cost Performance Index (CPI)

A measure of how close the project is to spending on the work performed to what was planned to

have been spent.

$$
{ \bf C P } | = \mathsf { E V } / \mathsf { A C }
$$

## INDEX VALUES

< 1: over budget or behind schedule

> 1: under budget or ahead of schedule
