## WBS Completion Criteria

 1.Status and completion are measurable.

2.The activity is bounded.

3.The activity has a deliverable.

4.Time and cost are easily estimated.

 5.Activity duration is within acceptable limits.

6.Work assignments are independent.

## Lesson Four: Plan Software Project

## Building the Initial Dependency Diagram

1. Enter task name and duration into software tool

2. Print PERT Diagram

3. Cut out each task node and affix to a Sticky Note

4. Write task ID # on Post-It Note

5. Place Sticky Notes on right side of whiteboard

6. Position start node on left side of whiteboard

7. Move all tasks with no predecessor to left side and connect to start node

8. Move all tasks on the right side with predecessors on the left side to the left side and connect with single line 9. Continue until all task nodes have been moved to left side
